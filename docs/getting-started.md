# Getting Started

## System Requirements
- iOS 14.0 or later
- Device Storage: 100 MB
- RAM: 2 GB minimum

## Installation Steps

1. Download the app from the App Store
2. Open the app
3. Complete the setup wizard
4. Start using ContestPilot

## First Time Setup

When you first launch the app:
1. Allow requested permissions
2. Set up your profile
3. Configure preferences
4. You're ready to go!
