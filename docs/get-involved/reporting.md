## How to Report Issues

Please email reports to [info@contestpilot.app](mailto:info@contestpilot.app) using the template below. Attach screenshots and, where relevant, exported logs.

### Report Template

**Environment**
- Device: (e.g., iPhone 14 Pro)
- OS version: (e.g., iOS 17.3)
- App version/build: (from TestFlight)
- Contest definition: (name/sponsor)

**Steps to Reproduce**
1. …
2. …
3. …

**Expected**
- …

**Actual**
- …

**Attachments**
- Screenshots: add `.zoomable` images; include validation messages if shown.
- Exports: ADIF/Cabrillo files via Share Sheet.

## What Makes a Great Report

- Clear, numbered steps and concise descriptions.
- Context: which contest definition and settings you used.
- Visuals: focused screenshots using app views referenced in the User Manual.

## Privacy & Data

- Do not include sensitive personal information in reports.
- QSO data shared is used solely for debugging and improving the app.
