## Eligibility & Scope

- Broader device support across recent iPhone/iPad models.
- Wider contest definitions and real‑world operating scenarios.

## Joining Beta

1. Install TestFlight from the App Store.
2. Request access via [info@contestpilot.app](mailto:info@contestpilot.app) or accept invite if already approved.
3. Install ContestPilot; confirm build/version shown in TestFlight.

## Focus Areas

- End‑to‑end workflows: setup → logging → scoring → export.
- Reliability under longer sessions and larger logs.
- Usability on iPad layouts (tables, split views).

## Feedback cadence

- Weekly summaries: notable successes and pain points.
- Immediate reports for crashes, data loss, or export errors.
