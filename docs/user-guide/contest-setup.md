- Browse grouped definitions or search by name/sponsor.
- Each definition encodes bands, modes, segments, and exchange rules.

## Contest Details

- Title: Any helpful name for your reference.
- Start Time (UTC): Used for scoring windows and logs.
- Operator: Callsign, grid/locator style, and default exchange.
