## Overview

ContestPilot is an amateur radio contest logger for iPhone and iPad. It streamlines contest setup, QSO entry, scoring, and export.

## Installation

- Download ContestPilot from the App Store when available.
- On first launch, you’ll see the splash screen and a short introduction.

## First Launch and Setup

- Open Settings to add your operator details: callsign, name, QTH, locator.
- Set your default exchange format if your contests typically use one.

## Core Concepts

- Contest: The container for your QSOs, rules, and scoring.
- QSO: Each contact you log (callsign, band/mode, exchange, time).
- Exchange: The information you send/receive (e.g., serial, grid, state).
- Rules: Contest‑specific validation and scoring handled by the built‑in rules engine.

## Create Your First Contest

1. Tap “New Contest”.
2. Choose a definition from the built‑in list (search or browse by sponsor).
3. Enter a title, start time (UTC), and confirm operator info.
4. Tap Create to open the contest dashboard.
