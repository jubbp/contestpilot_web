## Dashboard and Entry Fields

- Callsign: Enter the station callsign.
- Frequency → Band: Frequency infers band; set mode (SSB/CW/FT8, etc.).
- Signal Reports: RST or mode‑specific reports as required.
- Exchange: Serial or contest‑specific fields validated live.
- Log Button: Adds the QSO to the list with timestamp.

## Validation Feedback

- Duplicates/Reworks: Clear alerts if the contact is a dupe or rework.
- Band/Mode Restrictions: Warns if not allowed by the definition.
- Exchange Checks: Highlights missing or malformed fields.
