# Glossary

**ADIF (Amateur Data Interchange Format)**
Standard file format for exchanging amateur radio logs; supported in Export.

**Band**
The RF band inferred from frequency (e.g., 80m, 40m) and used for scoring/dupe checks.

**Cabrillo**
Log submission format commonly required by contest organizers; available in Export.

**Contest**
Your active log container that holds QSOs, scoring, and settings for a specific event.

**Contest Definition**
Built-in ruleset describing allowed bands/modes, exchanges, multipliers, and scoring.

**Duplicate (Dupe)**
A QSO that repeats the same station on the same band/mode where rework is not allowed.

**Exchange**
Information sent/received in a QSO (e.g., serial, grid, state); validated per contest rules.

**Grid/Locator**
Maidenhead locator or similar position code for the operator or exchanged data.

**Log**
Collection of QSOs for a contest; can be browsed, edited, and exported.

**Mode**
Operating mode (e.g., SSB, CW, FT8); may affect points, multipliers, or validity.

**Multiplier**
Scoring factor awarded for unique entities (states, grids, prefixes, segments) per rules.

**Rework**
A repeat contact that is allowed under certain conditions (e.g., on another band/mode or time window).

**Score Bar**
On-screen indicator showing current total points and multipliers as you log QSOs.

**Segment**
A rule-defined slice (e.g., band/mode/time category) that can affect scoring or multipliers.

**Share Sheet**
System UI used to export ADIF/Cabrillo files to other apps or destinations.

**Start Time (UTC)**
Contest start timestamp in UTC; used for timing, logging, and some scoring windows.

**Validation**
Real-time checks for exchange completeness, duplicates/reworks, and band/mode legality.

**QSO (Contact)**
An individual contact record with callsign, band/mode, exchange, time, and scoring impact.
