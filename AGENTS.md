## Project Overview

This is a documentation website for a IOS/MACOS application called ContestPilot

The site is hosted at contestpilot.app, which is a github pages site.

The site uses a Jekyll template called "just the docs"

Please limit token usage. Unless asked by the user, don't build or test the results. Ask the user to test for you.
